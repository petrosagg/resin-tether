resin-tether
=========

Resin container that tethers the Ethernet connection over Wifi.

## Environment Variables

* `SSID`
	* The SSID of your access point
	* Default: `ResinAP`
* `PASSPHRASE`
	* The Passphrase of your access point. It must be at least 8 characters.
	* Default: `12345678`
